# 欢迎页功能说明

## 概述

新增的欢迎页是金融平台的首页，为未登录用户提供了一个专业、美观的平台介绍页面。

## 功能特性

### 1. 页面结构

欢迎页包含以下主要区域：

#### 头部区域
- Logo图标（💼）
- 平台名称："金融平台"
- 英文副标题："Finance Platform"
- Logo具有浮动动画效果

#### Hero区域
- 主标题："智能投资管理，尽在掌握"
- 描述文字："专业的金融投资管理工具，帮助您轻松追踪和管理投资组合"
- 两个操作按钮：
  - **开始使用**：跳转到登录页
  - **了解更多**：平滑滚动到特性展示区域

#### 核心功能展示
6个特性卡片，展示平台的核心能力：
- 📊 **投资看板**：实时查看投资组合表现，直观的数据可视化
- 📈 **股票管理**：轻松记录和管理股票投资，追踪每笔交易的盈亏
- 💰 **收益分析**：全面分析投资收益，帮助做出更明智的决策
- 🔒 **安全保障**：银行级安全加密，保护个人信息和投资数据
- 📱 **多端同步**：支持多设备访问，随时随地管理投资组合
- 🎨 **主题定制**：支持深色/浅色主题切换，个性化使用体验

#### 统计数据展示
- 10,000+ 活跃用户
- ¥50亿+ 管理资产
- 99.9% 系统稳定性
- 24/7 技术支持

#### 底部区域
- 版权信息："© 2026 金融平台. 专业的投资管理工具"
- 功能链接：关于我们、隐私政策、使用条款

### 2. 交互功能

#### 按钮交互
- **开始使用**按钮：点击后跳转到登录页（/login）
- **了解更多**按钮：点击后平滑滚动到特性展示区域
- 所有按钮都有悬停效果和过渡动画

#### 卡片悬停效果
- 特性卡片在鼠标悬停时会上浮并显示阴影
- 图标会轻微放大，提供良好的交互反馈

### 3. 主题支持

欢迎页完全支持深色/浅色主题切换：

#### 浅色主题
- 背景：紫色到蓝紫色的渐变（#667eea → #764ba2）
- 卡片背景：白色半透明效果
- 文字：深色系

#### 深色主题
- 背景：深蓝色到藏青色的渐变（#1a1a2e → #16213e）
- 卡片背景：深色半透明效果
- 文字：浅色系

主题会自动保存到 localStorage，刷新后保持用户选择。

### 4. 响应式设计

页面完全响应式，适配不同屏幕尺寸：

- **桌面端**（> 768px）：3列网格布局展示特性卡片
- **平板端**（≤ 768px）：单列布局，字体和间距适当调整
- **移动端**（≤ 480px）：优化的移动体验，按钮全宽显示

### 5. 路由配置

#### 路由规则
- 访问根路径 `/` 时：
  - 未登录用户：显示欢迎页
  - 已登录用户：自动重定向到投资看板（/dashboard）
  
- 从欢迎页可以导航到：
  - `/login`：登录页

#### 路由保护
- 欢迎页不需要认证即可访问
- 已登录用户访问欢迎页会自动跳转到投资看板

### 6. 技术实现

#### 使用的技术栈
- Vue 3 Composition API
- Vue Router 4
- Naive UI 组件库
- CSS3 动画和过渡效果

#### 主要组件
- WelcomePage.vue：欢迎页主组件
- 集成了 useTheme composable 支持主题切换

#### 动画效果
- Logo浮动动画
- 按钮悬停动画
- 卡片悬停效果
- 平滑滚动效果
- 页面过渡动画

## 使用方法

### 访问欢迎页
1. 在浏览器中访问应用根路径：`http://localhost:3000/`
2. 如果未登录，会显示欢迎页
3. 如果已登录，会自动跳转到投资看板

### 导航到登录页
点击"开始使用"按钮即可跳转到登录页

### 了解平台功能
点击"了解更多"按钮会平滑滚动到特性展示区域
